# Club Placement App

A web application built for Plantation Park Elementary School to automate the after school enrichment club assignment process.

## The Problem
Every session, staff manually processed paper forms, ran a random lottery, and hand-wrote acceptance letters for hundreds of students. The process was time-consuming and error-prone.

## The Solution
This app digitizes the entire workflow:
- Parents submit club choices via an online form
- The app validates grade-appropriate club selections
- A randomized family lottery assigns students to clubs
- Acceptance letters and attendance sheets are generated automatically

## Features
- [ ] Club setup and management
- [ ] Student data import from Google Forms
- [ ] Grade-level validation
- [ ] Family-based lottery randomizer
- [ ] Automatic club assignment with waitlist handling
- [ ] Acceptance letter generation
- [ ] Attendance sheet generation
